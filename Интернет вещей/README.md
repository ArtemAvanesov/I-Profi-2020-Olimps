В этой системе имеется аналоговый датчик температуры LM35, который имеет следующую спецификацию:
напряжение питания: от 2.7В до 5.5В;

точность: ± 0.5∘C;
диапазон измерений: -55∘C to +150∘C (макс. +150∘C);
